In Angular, @Input() decorator used to pass data from a parent component to child component.It allows a child component to receive data from its parent through property binding.

  1. import [ Input ] from '@angular/core'; in CHILD Component
  2. Add this decorator in CHILD Component - @Input() message: string | undefined; 
    
                @Input()  \\ this is decorator
                message \\ this is we getting from PARENT Component
                string | undefined \\ this the data type
  3. Add this interpolation [[message]] - in CHILD Component HTML
  4. PARENT Component
    Sending data to CHILD from PARENT Component
