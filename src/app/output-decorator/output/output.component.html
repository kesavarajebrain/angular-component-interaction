Output

In Angular, @Output() decorator used to pass data from a CHILD component to PARENT component. It allows child components to notify their parents about changes or actions.This is achieved using an EventEmitter.

This is parent component (Output component is PARENT)

Here we created child component!

      ⮜app-daughter⮞⮜/app-daughter⮞

Technical Breakup:

Implementing @Output() Decorator

  1. import [ Output ] from '@angular/core'; in CHILD Component
  2. import [ EventEmitter ] from '@angular/core'; in CHILD Component
  3. Add this decorator in CHILD Component
    SYNTAX:
    @Output() sendEventToParent = new EventEmitter(); 
      
                  @Output()  \\ this is decorator
                  sendEventToParent \\ this is we emit from CHILD
                  new EventEmitter() \\ this is we utilize for emit data
  4. Here we passing input value from CHILD to PARENT
    CHILD Component 
    
                child_message: any;

                sendToParent() [
                    this.sendEventToParent.emit(this.child_message);  // here we emit child message to parent
                  ]
  5. PARENT Component
    Sending data to PARENT from CHILD Component
    SYNTAX:
    ⮜parent-component-div⮞
    ⮜app-daughter (sendEventToParent)="receiveMsgFrmChild($event)" ⮞⮜/app-daughter⮞
    ⮜/parent-component-div⮞ 
      
                (sendEventToParent)  // this is emit from Child - Used in (CHILD Component) 
                receiveMsgFrmChild  // this is we receive data in Parent - Used in (PARENT Component)
                ($event)  // this also we receive data in Parent if any parameter comes - Used in (PARENT Component)
  6. For above example we are sending value from CHILD component via clicking the Send to parent button.
    SYNTAX:
    
                childMessage: string | undefined;

                receiveMsgFrmChild(value: string) [
                  this.childMessage = value; // here we get the value from CHILD and Interpolate in PARENT HTML
                  console.log(value);
                ]
