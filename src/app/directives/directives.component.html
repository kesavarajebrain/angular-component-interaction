Directives

A Directive is a class that can change the appearance, behavior, or layout of DOM elements in an Angular application.

üëâ You can think of directives as instructions to the DOM.

There are 3 main types:
Type Description Example
1.Component Directive A directive with a **template** `@Component( )`
2.Structural Directive Changes the **structure** of the DOM (adds/removes elements) `*ngIf`, `*ngFor`, `*ngSwitch`
3.Attribute Directive Changes the **appearance or behavior** of an existing element `ngClass`, `ngStyle`, `ngModel`
