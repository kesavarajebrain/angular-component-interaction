1️⃣ resource() – Signals for Async Data (HTTP)

Problem Before Signals
Fetching API data in Angular traditionally:

this.http.get('/api/tasks').subscribe(tasks => { this.tasks = tasks; }); 
Problems:
manual subscribe
manual loading state
manual error handling
unsubscribe issues
With resource()

Angular created resource() to manage async data with signals.

    It automatically handles:
  • loading
  • error
  • data
  • re-fetch
